#红外控制模块服务 piInfrated
 * 本模块是一个服务，专门用户接受客户端发送的遥控命令，可以控制电视，空调
 * 响应移动的号召，加入web控制界面[web 红外控制界面](https://github.com/apanly/RemoterUI)  前端界面使用zmq与此模块服务器通信
 * 本模块会是一个单独的红外控制模块，会和我的[piRobot](https://github.com/apanly/piRobot)结合，用于语音控制遥控器

#python依赖包
* pyzmq
